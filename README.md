# SpringMvc-dubbo-MyBatis

> 1.基于spring4.3.2实现，集成Dubbo Zookeeper MyBatis Spring SpringMvc ..

> 2.服务组件为可执行jar包






## 运行方式

> cd SpringMvc-dubbo-MyBatis

> mvn clean install 

> cd dsp-cache**/target

> java -jar dsp****one**jar
  
> cd ../dsp-user**/target

> java -jar dsp****one**jar
